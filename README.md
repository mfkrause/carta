# oxidoc

A clean-room reimplementation of [pandoc](https://pandoc.org) in Rust.

Goals: smaller binaries, better performance, compile-time memory safety, and a smoother developer
experience.

**Status:** early scaffolding. See [`docs/PORTING.md`](docs/PORTING.md) for the architecture and
roadmap.

This is an independent project. It is not affiliated with pandoc, contains no pandoc source code,
and is built clean-room from public format specifications and observable behavior.
