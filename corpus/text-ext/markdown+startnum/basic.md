An ordered list keeps the number it is written to open at. This list starts at three, so its items
are numbered three, four, and five rather than being renumbered from one:

3. third
4. fourth
5. fifth

The starting value is taken from the first marker alone; the later markers only advance the counter
by one apiece, so a list whose written numbers jump around still counts up from its opening value:

7. seventh
9. eighth
11. ninth

The delimiter style is preserved alongside the start, so a list opened with a parenthesis keeps both:

4) fourth
5) fifth

A list that happens to open at one is numbered from one, exactly as an unmarked list would be:

1. first
2. second
